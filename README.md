# bio-link

A lightweight personal link-in-bio page for Pietro Lungarini.

Built as a static GitHub Pages site with a clean mobile-first layout, theme switching, social links, a GitHub profile preview, and support for multiple pages as the site grows.

## Live site

https://bio.pietro.lungarini.it

## Pages

- `/` — main bio link page
- `/evenhub-apps/` — Even Realities smart glasses apps

## Stack

- HTML
- CSS
- Vanilla JavaScript
- GitHub Pages

## Development

This repo uses a simple static setup. No build step and no dependencies.

Open locally with any static server:

```bash
npx serve .
```

You can also open `index.html` directly in the browser for quick checks.

## Deployment

The site is deployed through GitHub Pages from the `master` branch.

Custom domain:

```txt
bio.pietro.lungarini.it
```
